Tema 1: Elementos de Medicion

En HTML5 se incorporaron dos nuevas etiquetas para mostar informacion de manera estadistica.

Estos elementos son:

<progres>(indica cuanto llevamos realizado de una tarea, mientras que esta aun no este finalizada, se implemeta tanto para barras de carga, como para indicar un avance respecto a cualquier tipo de actividad)


<meter>(indica los valores correspondientes a una actividad finalizada, como una votacion o la conclusion de un examen. A diferencia de progres se pueden establecer diferentes caracteristicas en base a los valores optimos y bajos)
